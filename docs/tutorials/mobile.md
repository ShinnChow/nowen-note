# 在手机上使用 nowen-note

> 在 Android 手机上安装和使用 nowen-note，随时随地记笔记。

---

## 安装方式

### 方式一：下载 APK

1. 访问 [Releases 页面](https://github.com/cropflre/nowen-note/releases)
2. 下载最新版 `.apk` 文件
3. 在手机上安装（需允许安装未知来源应用）

### 方式二：手机浏览器

直接在手机浏览器中访问你的 nowen-note 服务器地址，例如 `http://192.168.1.100:3001`。

---

## 移动端界面

移动端采用抽屉式布局：

- **默认**：显示编辑器
- **左滑/点 ☰**：打开侧边栏
- **点笔记**：切换笔记

[截图：移动端侧边栏]

---

## 移动端操作

| 操作 | 方法 |
|---|---|
| 打开侧边栏 | 点击左上角 ☰ 菜单按钮 |
| 新建笔记本 | 侧边栏顶部 ＋ 按钮 |
| 新建笔记 | 选笔记本 → 新建笔记 |
| 更多操作 | 长按笔记本或笔记 |
| 移动笔记 | 长按笔记 → 移动到… |
| 搜索 | 侧边栏顶部搜索框 |

[截图：长按弹出菜单]

---

## 移动端编辑

编辑器在手机上完整可用：

- 点击正文区域弹出键盘
- 格式工具栏在键盘上方
- 支持所有富文本格式
- 支持图片粘贴和拖入
- 自动保存

---

## 常见问题

### Q：手机上打不开侧边栏？

确认是点击 ☰ 按钮，不是滑动。侧边栏以抽屉形式从左侧滑出。

### Q：编辑时键盘遮挡内容？

编辑器会自动滚动让光标可见。如果仍有问题，可以收起侧边栏获得更多编辑空间。

### Q：移动端和桌面端数据同步吗？

是的。登录同一账号，数据通过服务器实时同步。
