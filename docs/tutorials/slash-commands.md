# 斜杠命令和快捷操作

> 用斜杠命令和快捷键快速插入各种格式。

---

## 斜杠命令

在编辑器新行输入 `/` 会弹出命令菜单，可以快速插入各种格式和元素。

[截图：斜杠命令菜单]

---

## 命令分类

### 标题类

| 命令 | 快捷键 | 效果 |
|---|---|---|
| `/标题1` | `Ctrl/Cmd + 1` | 一级标题 |
| `/标题2` | `Ctrl/Cmd + 2` | 二级标题 |
| `/标题3` | `Ctrl/Cmd + 3` | 三级标题 |

### 列表类

| 命令 | 效果 |
|---|---|
| `/列表` | 无序列表 |
| `/有序` | 有序列表 |
| `/待办` | 待办事项列表 |

### 格式类

| 命令 | 效果 |
|---|---|
| `/引用` | 引用块 |
| `/代码` | 代码块（选择语言） |
| `/Mermaid` | Mermaid 图表 |
| `/数学` | 数学公式 |
| `/脚注` | 脚注 |
| `/分割线` | 水平分割线 |

### 行内格式

| 命令 | 快捷键 | 效果 |
|---|---|---|
| `/加粗` | `Ctrl/Cmd + B` | **加粗** |
| `/斜体` | `Ctrl/Cmd + I` | *斜体* |
| `/高亮` | — | 高亮 |
| `/删除线` | `Ctrl/Cmd + Shift + S` | ~~删除线~~ |
| `/行内代码` | `Ctrl/Cmd + E` | `代码` |
| `/链接` | `Ctrl/Cmd + K` | 超链接 |

### 插入类

| 命令 | 效果 |
|---|---|
| `/图片` | 插入图片 |
| `/视频` | 插入视频 |
| `/表格` | 插入表格 |

### AI 类

| 命令 | 效果 |
|---|---|
| `/AI` | 打开 AI 写作助手 |

---

## 常用快捷键

| 快捷键 | 效果 |
|---|---|
| `Ctrl/Cmd + B` | 加粗 |
| `Ctrl/Cmd + I` | 斜体 |
| `Ctrl/Cmd + E` | 行内代码 |
| `Ctrl/Cmd + K` | 插入链接 |
| `Ctrl/Cmd + Shift + S` | 删除线 |
| `Ctrl/Cmd + H` | 搜索替换 |
| `Tab` | 增加缩进 |
| `Shift + Tab` | 减少缩进 |
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Shift + Z` | 重做 |

---

## 技巧

### 搜索命令

在斜杠菜单中输入关键词可以过滤命令。例如输入 `/表` 会显示表格相关的命令。

### 快速触发

不需要打完整的命令名。例如 `/h1`、`/h2`、`/h3` 也能触发标题命令。
