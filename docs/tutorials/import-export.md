# 笔记导入和导出

> 从其他应用迁移笔记，或将笔记导出备份。

---

## 导入

### 从 Word 文档导入

1. 在笔记列表中右键笔记本
2. 选择「导入 Word 文档」
3. 选择 .docx 文件
4. 内容会自动转换为 nowen-note 格式

### 从公众号文章导入

1. 复制微信公众号文章链接
2. 右键笔记本 →「导入公众号文章」
3. 粘贴链接
4. 文章内容会自动导入

### 从 URL 导入

1. 右键笔记本 → 选择 URL 导入
2. 输入网页链接
3. 内容会被抓取并创建为笔记

### 从其他笔记应用迁移

在 ⚙️ 设置 → 数据管理 → 导入中，支持：

| 应用 | 说明 |
|---|---|
| 有道云笔记 | 导出为特定格式后导入 |
| 小米笔记 | 导出后导入 |
| iCloud 备忘录 | 导出后导入 |
| OPPO 便签 | 导出后导入 |

---

## 导出

### 单篇笔记导出

1. 在笔记列表中右键笔记
2. 选择导出格式：

| 格式 | 说明 |
|---|---|
| Markdown | 导出为 .md 文件 |
| PDF | 导出为 PDF |
| 图片 | 导出为 PNG 图片 |
| Word | 导出为 .docx 文件 |

### 整个笔记本导出

1. 右键笔记本
2. 选择「导出为 Markdown」
3. 该笔记本下所有笔记会导出为 Markdown 文件

### 导出为 ZIP

在设置 → 数据管理中可以导出完整数据为 ZIP 包。

---

## 常见问题

### Q：导入后格式变了？

不同应用的格式有差异。导入后建议检查格式，必要时手动调整。

### Q：导入的图片在哪？

导入的图片会自动保存到附件系统中。

### Q：导出的 Markdown 可以在其他应用打开吗？

可以。标准 Markdown 格式，兼容所有支持 Markdown 的编辑器。
