# 安全设置

> 保护你的 nowen-note 账号：修改密码、启用 2FA、管理登录。

---

## 进入安全设置

1. 点击左侧导航栏 ⚙️ 设置
2. 选择「安全设置」

[截图：安全设置入口]

---

## 修改密码

1. 在安全设置中找到「修改密码」区域
2. 输入当前密码
3. 输入新密码（至少 6 位）
4. 确认新密码
5. 点击保存

> ⚠️ 首次登录后请立即修改默认密码！

---

## 两步验证（2FA）

两步验证为你的账号增加一层保护。启用后，登录时除了密码还需要输入手机验证器的动态码。

### 启用 2FA

1. 在安全设置中找到「两步验证」区域
2. 点击「启用」
3. 用手机验证器 App（如 Google Authenticator、Authy）扫描二维码
4. 输入 App 显示的 6 位验证码
5. 系统会显示恢复码，**请妥善保存**

[截图：2FA 设置界面]

### 使用 2FA 登录

1. 输入用户名和密码
2. 打开验证器 App
3. 输入 6 位动态码

### 关闭 2FA

1. 安全设置 → 两步验证 →「关闭」
2. 输入当前动态码确认

### 恢复码

启用 2FA 时系统会生成恢复码。如果手机丢失，可以用恢复码登录。

> ⚠️ 恢复码只能使用一次。建议保存在安全的地方。

## 会话管理

安全设置中可以查看当前登录的所有设备和会话。

### 查看活跃会话

显示所有当前登录的设备、浏览器、IP 地址和最后活跃时间。

### 踢出设备

点击某个会话旁的「退出」按钮，可以远程注销该设备。

---

## API Token

API Token 用于外部工具（如浏览器剪藏扩展、MCP Server、SDK）访问 nowen-note。

### 创建 Token

1. 在设置中找到「API Token」
2. 点击「新建 Token」
3. 设置名称和权限范围
4. 复制生成的 Token（只会显示一次）

### 使用 Token

外部工具通过 HTTP Header `Authorization: Bearer <token>` 认证。

---

## 体验账号限制

如果账号被标记为体验账号（`isDemo`），以下功能受限：

- 无法修改密码
- 无法修改用户名
- 无法启用/关闭 2FA
