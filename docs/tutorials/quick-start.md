# 5 分钟快速上手 nowen-note

> 从零开始，5 分钟内创建你的第一篇笔记并体验 AI 功能。

---

## 1. 安装 nowen-note

nowen-note 支持多种安装方式，选择最适合你的：

### 方式一：Docker 一键部署（推荐）

```bash
git clone https://github.com/cropflre/nowen-note.git
cd nowen-note
docker-compose up -d
```

安装完成后，浏览器访问 `http://<你的IP>:3001`。

### 方式二：本地开发

需要 Node.js 20+。

```bash
git clone https://github.com/cropflre/nowen-note.git
cd nowen-note
npm run install:all
npm run dev:backend   # 后端 :3001
npm run dev:frontend  # 前端 :5173
```

浏览器访问 `http://localhost:5173`。

### 方式三：在线体验

不想部署？直接打开 [https://note.nowen.cn/](https://note.nowen.cn/)。

- 账号：`demo`
- 密码：`demo123456`

> ⚠️ 体验账号为只读演示，数据可能被定期重置。

更多部署方式请参考 [部署指南](../deployment.md)。

打开 nowen-note 后，你会看到登录页面。

[截图：登录页面]

| 字段 | 值 |
|---|---|

## 3. 创建你的第一个笔记本

登录后，你会看到三栏布局的主界面：

- **左侧**：导航栏 + 文档树（笔记本）
- **中间**：笔记列表
- **右侧**：编辑器

[截图：nowen-note 主界面标注]

**创建笔记本：**

1. 看左侧边栏「笔记本」区域上方
2. 点击 **＋** 按钮
3. 输入名称，例如「我的笔记」
4. 按回车确认

[截图：创建笔记本]

> 💡 **小提示**：笔记本就像文件夹，用来分类管理笔记。你可以创建多个笔记本，比如「工作」、「学习」、「生活」。

---

## 4. 创建你的第一篇笔记

1. 点击左侧刚创建的笔记本
2. 中间区域会出现「新建笔记」按钮
3. 点击后，右侧编辑器会打开
4. 输入标题和内容
5. 内容会**自动保存**

[截图：创建笔记并编辑]

> 💡 **自动保存**：nowen-note 会自动保存你的编辑内容，不需要手动按 Ctrl+S。编辑器底部会显示保存状态。

---

## 5. 体验富文本编辑

nowen-note 的编辑器支持丰富的格式：

- **斜杠命令**：在编辑器中输入 `/`，会弹出命令菜单
- **格式工具栏**：选中文字后，上方会出现格式工具栏
- **拖入图片**：直接把图片拖进编辑器

### 常用斜杠命令

| 命令 | 效果 |
|---|---|
| `/标题1` | 插入一级标题 |
| `/标题2` | 插入二级标题 |
| `/列表` | 插入无序列表 |
| `/有序` | 插入有序列表 |
| `/待办` | 插入待办事项 |
| `/代码` | 插入代码块 |
| `/表格` | 插入表格 |
| `/图片` | 插入图片 |
| `/数学` | 插入数学公式 |
| `/Mermaid` | 插入流程图 |
| `/AI` | AI 写作助手 |

## 6. 体验 AI 功能

### 6.1 配置 AI（可选）

nowen-note 支持多个 AI 服务商，你需要至少配置一个才能使用 AI 功能。

1. 点击左侧导航栏的 ⚙️ 设置
2. 选择「AI 设置」
3. 选择一个 AI 服务商
4. 填入 API Key
5. 选择模型

[截图：AI 设置面板]

**支持的 AI 服务商：**

| 服务商 | 说明 |
|---|---|
| 通义千问 | 阿里云，国内访问快 |
| OpenAI | GPT-4o / GPT-4 |
| Google Gemini | Google AI |
| DeepSeek | DeepSeek-V3 / R1 |
| 豆包 | 字节跳动 |
| Ollama | 本地部署，无需 API Key |

### 6.2 使用 AI 生成标题

1. 写完一篇笔记
2. 点击编辑器工具栏的 AI 图标
3. 选择「生成标题」
4. AI 会根据内容自动生成标题

[截图：AI 生成标题]

### 6.3 使用 AI 推荐标签

1. 在编辑器工具栏点击 AI 图标
2. 选择「推荐标签」
3. AI 会分析内容并推荐合适的标签

### 6.4 使用 AI 总结

1. 在编辑器工具栏点击 AI 图标
2. 选择「AI 总结」
3. AI 会生成内容摘要
4. 可以选择追加到笔记末尾

## 7. 创建更多笔记本和笔记

现在你已经掌握了基本操作，可以开始组织你的笔记了：

1. **创建多个笔记本**：为不同主题创建笔记本
2. **创建子笔记本**：右键笔记本 →「新建子笔记本」
3. **拖拽排序**：拖动笔记本调整顺序
4. **更换图标**：右键笔记本 →「更换图标」→ 选择 Emoji

### 推荐的笔记本结构

```
工作
├── 项目 A
├── 项目 B
└── 会议记录

学习
├── 编程
├── 读书笔记
└── 英语

生活
├── 旅行计划
├── 健康记录
└── 食谱收藏
```
