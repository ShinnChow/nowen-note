# 数据同步常见问题

> 解决多端同步、离线编辑、冲突等常见问题。

---

## 同步机制

nowen-note 的数据同步基于 WebSocket 实时通信：

- 编辑内容自动保存并同步到服务器
- 服务器推送给其他连接的设备
- 断网时数据暂存本地，联网后自动同步

---

## 常见问题

### Q：手机上修改的笔记，电脑上看不到？

1. 确认两端登录同一账号
2. 确认连接同一服务器
3. 刷新页面
4. 检查网络连接

### Q：修改后另一个设备没更新？

实时同步通常在 1-2 秒内完成。如果延迟较长：

1. 检查网络连接
2. 确认 WebSocket 连接正常（浏览器开发者工具 → Network → WS）
3. 尝试刷新页面

### Q：离线编辑后数据丢失了？

不会。离线编辑的内容保存在本地 localStorage 中。联网后会自动同步。

但如果清除了浏览器数据（localStorage），离线内容会丢失。

### Q：出现「版本冲突」怎么办？

1. 选择「重新加载」— 用服务器版本覆盖本地
2. 选择「保留本地」— 以本地版本为准

建议在冲突前先同步（确保显示「已保存」）。

### Q：多端同时编辑同一篇笔记？

可以。nowen-note 支持实时协作编辑，多人可以同时编辑同一篇笔记。

---

## 数据一致性

### Q：附件（图片）在多端都能看到吗？

是的。附件存储在服务器上，所有设备通过 URL 访问。

### Q：离线时上传的图片呢？

离线时图片会暂存本地。联网后自动上传到服务器。

| 收藏状态 | ✅ 实时同步 |
| 附件文件 | ✅ 上传到服务器 |
| 思维导图 | ✅ 实时同步 |
| 说说 | ✅ 实时同步 |
| 待办任务 | ✅ 实时同步 |
| 用户设置 | ✅ 登录时同步 |
