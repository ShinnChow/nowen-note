# 浏览器剪藏扩展

> 用浏览器扩展一键保存网页内容到 nowen-note。

---

## 什么是剪藏？

剪藏扩展可以把你正在浏览的网页内容直接保存到 nowen-note 中作为一篇笔记。支持 Chrome 和 Edge 浏览器。

---

## 安装扩展

### Chrome

1. 访问 [Chrome Web Store](https://chromewebstore.google.com/detail/nowen-note-web-clipper/nglkodhfdbnfielchjpkjhenfaecafpg)
2. 点击「添加到 Chrome」

### Edge

1. 在 Edge 扩展商店搜索 nowen-note
2. 或从 Chrome Web Store 安装（Edge 兼容 Chrome 扩展）

---

## 配置扩展

首次使用需要配置：

1. 点击工具栏上的 Nowen 图标
2. 弹出「未配置」提示 → 打开设置
3. 填写服务器地址：`https://note.example.com` 或 `http://localhost:3001`
4. 填写 API Token（在 nowen-note 设置 → API Token 中创建）
5. 点击「测试连接」，看到 ✅ 即可

---

## 使用方法

### 方法一：弹窗剪藏

1. 在任意网页上点击扩展图标
2. 选择剪藏模式
3. 点击「剪藏」

### 方法二：右键菜单

1. 在网页上右键
2. 选择「剪藏到 nowen-note」

### 方法三：快捷键

| 快捷键 | 效果 |
|---|---|
| `Alt + Shift + S` | 剪藏整页 |
| `Alt + Shift + A` | 剪藏选区 |

---

## 剪藏模式

| 模式 | 说明 |
|---|---|
| 正文 | 自动提取网页正文（Readability） |
| 选区 | 只剪藏你选中的内容 |
| 简化 | 提取正文并简化格式 |
| 整页 | 保存完整网页 |
| 截图 | 截取当前可见区域 |
| 完整截图 | 截取整个网页 |

---

## AI 增强

剪藏后可以调用 AI 自动处理内容：

| 功能 | 说明 |
|---|---|
| 生成摘要 | AI 自动生成内容摘要 |
| 生成大纲 | AI 生成结构化大纲 |
| 推荐标签 | AI 分析内容推荐标签 |
| 重写标题 | AI 生成更好的标题 |
| 重点高亮 | AI 标记重要内容 |
| 翻译 | AI 翻译为其他语言 |

AI 增强使用 nowen-note 服务器配置的 AI 服务，扩展本身不持有 API Key。

---

## 图片处理

剪藏时的图片处理方式：

| 方式 | 说明 |
|---|---|
| 下载为本地 | 图片下载到服务器附件系统 |
| 保留原 URL | 保留网页上的原始图片链接 |
| 忽略图片 | 不保存图片 |

---

## 归类设置

可以指定剪藏的笔记保存到哪个笔记本，以及自动添加标签。

---

## 常见问题

### Q：剪藏按钮灰色？

确认已正确配置服务器地址和 API Token。

### Q：图片没保存？

检查图片处理方式设置。选择「下载为本地」才会保存图片。

### Q：剪藏失败？

1. 检查网络连接
2. 检查 API Token 是否有效
3. 检查服务器是否正常运行
