# 了解 nowen-note 界面

> 熟悉 nowen-note 的三栏布局，快速找到你需要的功能。

---

## 界面总览

nowen-note 采用经典的三栏布局：

```
┌──────┬──────────┬──────────────────────┐
│ 导航栏 │ 笔记列表  │       编辑器           │
│      │          │                      │
│ 📚   │ 笔记 1   │  标题                │
│ ⭐   │ 笔记 2 ● │                      │
│ 📁   │ 笔记 3   │  正文内容...          │
│ 🗑️   │          │                      │
│      │          │                      │
│ ──   │          │                      │
│ 📓   │          │                      │
│ 笔记本 │          │                      │
│  树   │          │                      │
└──────┴──────────┴──────────────────────┘
```

[截图：nowen-note 主界面标注]

## 左侧：导航栏

导航栏位于最左侧，提供全局导航入口。

### 导航项

| 图标 | 名称 | 说明 |
|---|---|---|
| 📚 | 所有笔记 | 查看所有笔记（不限笔记本） |
| ⭐ | 收藏 | 你收藏的笔记 |
| 📁 | 文件管理 | 管理上传的附件和文件 |
| 🗑️ | 回收站 | 已删除的笔记（可恢复） |
| 📝 | 说说 | 时间线式短内容 |
| ✅ | 待办 | 任务清单和看板 |
| 🧠 | 思维导图 | 节点式思维导图编辑器 |
| 💬 | AI 问答 | 基于笔记的 AI 知识问答 |

### 工作区切换

导航栏顶部有工作区切换器。点击可以：

- 在个人空间和工作区之间切换
- 创建新工作区
- 加入已有工作区（输入邀请码）

[截图：工作区切换器]

---

## 左侧：文档树（侧边栏）

导航栏右侧是侧边栏，显示文档树结构。

### 顶部区域

- **搜索框**：全局搜索笔记标题和正文
- **＋按钮**：创建新笔记本

### 笔记本树

以树形结构展示所有笔记本，支持：

- 点击切换笔记本
- ▸/▾ 折叠和展开子笔记本
- 右键菜单：新建子笔记本、重命名、删除、移动、更换图标
- 拖拽排序和移动

### 底部区域

- **标签**：显示所有标签，点击筛选带该标签的笔记

---

## 中间：笔记列表

选中笔记本后，中间区域显示该笔记本下的笔记列表。

### 列表项信息

每条笔记显示：

- 标题（搜索时高亮匹配文字）
- 内容摘要
- 更新时间
- 置顶/收藏标记

### 排序

点击列表顶部排序按钮切换排序方式：

- **手动排序**（默认）：拖拽调整顺序
- **按更新时间**
- **按创建时间**
- **按标题**

### 批量操作

按住 Ctrl/Cmd 点击多选笔记，或 Shift 点击范围选择。选中后右键可以：

- 批量移动
- 批量删除（移入回收站）
- AI 批量标签
- AI 批量归类

---

## 右侧：编辑器

点击笔记后，右侧打开编辑器。

### 编辑器布局

- **顶部工具栏**：格式按钮、AI 功能、笔记操作（收藏、置顶、锁定、分享）
- **标题栏**：笔记标题（点击编辑）
- **标签栏**：笔记标签（点击管理）
- **正文区域**：富文本编辑
- **底部状态栏**：保存状态、字数统计

### 工具栏常用按钮

| 按钮 | 功能 |
|---|---|
| ⭐ | 收藏/取消收藏 |
| 📌 | 置顶/取消置顶 |
| 🔒 | 锁定/解锁笔记 |
| 🗑️ | 移入回收站 |
| 🤖 | AI 功能菜单 |
| 📎 | 附件管理 |

---

## 可调节布局

拖拽栏之间的分隔线可以调整各区域宽度：

- 拖拽导航栏和笔记列表之间的线 → 调整侧边栏宽度
- 拖拽笔记列表和编辑器之间的线 → 调整列表宽度

侧边栏也可以完全收起，给编辑器更多空间。

---

## 移动端界面

在手机上，nowen-note 采用抽屉式布局：

- 默认显示编辑器
- 点击左上角 ☰ 打开侧边栏（抽屉滑出）
- 侧边栏中可以切换笔记本和笔记

[截图：移动端界面]

---

## 设置入口

点击导航栏底部的 ⚙️ 图标打开设置面板，包含：

- **AI 设置**：配置 AI 服务商
- **安全设置**：修改密码、2FA
- **用户管理**：管理用户（管理员）
- **数据管理**：备份、导入、导出
- **显示设置**：主题、字体、阅读密度
