# 思维导图模块入门

> 用节点式思维导图整理思路、梳理知识结构。

---

## 思维导图是什么？

思维导图是一种将想法以树状结构展示的工具。nowen-note 内置了思维导图编辑器，你可以直接在应用中创建和编辑思维导图，也可以从笔记一键生成。

---

## 创建思维导图

### 方法一：从导航栏进入

1. 点击左侧导航栏的「思维导图」图标
2. 点击「新建思维导图」
3. 在编辑器中开始添加节点

[截图：思维导图编辑器界面]

### 方法二：从笔记生成

1. 打开一篇笔记
2. 点击编辑器工具栏的 🤖 AI 图标
3. 选择「生成思维导图」
4. AI 会根据笔记内容自动生成思维导图

[截图：从笔记生成思维导图]

---

## 思维导图操作

### 基础操作

| 操作 | 方法 |
|---|---|
| 添加子节点 | 选中节点 → Tab 键 |
| 添加同级节点 | 选中节点 → Enter 键 |
| 编辑节点文字 | 双击节点 |
| 删除节点 | 选中节点 → Delete 键 |
| 拖拽移动节点 | 按住节点拖动 |

### 编辑器工具栏

- **撤销/重做**：左上角按钮
- **缩放**：鼠标滚轮或工具栏按钮
- **全屏**：工具栏全屏按钮
- **导出**：导出为 PNG 图片

---

## 思维导图保存

思维导图和笔记一样，编辑后会自动保存。你可以在「思维导图」列表中查看所有创建的思维导图。
