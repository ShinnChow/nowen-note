# 富文本编辑器使用教程

> 掌握 nowen-note 编辑器的所有编辑技巧，让笔记更美观、更高效。

---

## 编辑器概览

nowen-note 的编辑器基于 Tiptap 3 构建，提供所见即所得的富文本编辑体验。

[截图：编辑器全貌标注]

编辑器分为三个区域：

- **顶部工具栏**：格式按钮、AI 功能、笔记操作
- **正文区域**：输入和编辑内容
- **底部状态栏**：保存状态、字数统计、编辑器模式切换

---

## 基础编辑

### 输入文字

直接在编辑器中点击并输入文字。内容会**自动保存**，无需手动按 Ctrl+S。

### 标题

**快捷键：** `Ctrl/Cmd + 1`（一级）/ `2`（二级）/ `3`（三级）

**工具栏：** 选中文字 → 点击标题按钮 → 选择级别

**斜杠命令：** 输入 `/标题1`、`/标题2`、`/标题3`

---

## 文字格式

选中文字后，上方自动弹出格式工具栏：

| 格式 | 快捷键 | 效果 |
|---|---|---|
| **加粗** | `Ctrl/Cmd + B` | **加粗文字** |
| *斜体* | `Ctrl/Cmd + I` | *斜体文字* |
| ~~删除线~~ | `Ctrl/Cmd + Shift + S` | ~~删除线~~ |
| `行内代码` | `Ctrl/Cmd + E` | `行内代码` |
| 高亮 | — | 文字高亮显示 |
| 链接 | `Ctrl/Cmd + K` | 插入超链接 |

---

## 列表

| 类型 | 触发方式 | 说明 |
|---|---|---|
| 无序列表 | 输入 `-` + 空格 | 圆点列表 |
| 有序列表 | 输入 `1.` + 空格 | 数字编号列表 |
| 待办事项 | 输入 `/待办` | 可勾选的待办列表 |

列表支持缩进：按 Tab 键增加缩进，Shift + Tab 减少缩进。

---

## 引用块

输入 `>` + 空格，或输入 `/引用`。

> 这是一段引用文字，适合用来标注重点或引用他人内容。

---

## 代码块

输入 `/代码`，选择编程语言后编写代码。

支持语法高亮的语言包括：JavaScript、TypeScript、Python、Java、Go、Rust、HTML、CSS、SQL、Bash 等。

---

## 表格

输入 `/表格` 插入表格。

| 操作 | 方法 |
|---|---|
| 调整列宽 | 拖拽列边框 |
| 调整行高 | 拖拽行边框 |
| 插入行 | 右键单元格 → 插入行 |
| 删除行 | 右键单元格 → 删除行 |
| 合并单元格 | 选中多个单元格 → 右键合并 |

---

## 图片

**拖拽**：直接把图片拖进编辑器

**粘贴**：复制图片后 `Ctrl/Cmd + V`

**斜杠命令**：输入 `/图片`，选择本地图片上传

上传的图片自动保存到附件系统。点击笔记标题栏右侧的附件图标可查看和管理。

---

## 链接

选中文字 → `Ctrl/Cmd + K` → 输入链接地址

或输入 `/链接`

---

## 数学公式

输入 `/数学` 插入数学公式块，支持 LaTeX 语法：

- 行内：`$E = mc^2$`
- 块级：`$$\frac{-b \pm \sqrt{b^2-4ac}}{2a}$$`

---

## Mermaid 图表

输入 `/Mermaid` 插入图表块。

支持：流程图、时序图、类图、甘特图、状态图。

---

## 脚注

输入 `/脚注` 插入脚注引用，自动编号并在页面底部显示完整内容。

---

## 分割线

输入 `---` 或 `/分割线` 插入水平分割线。

---

## 斜杠命令完整列表

在编辑器中输入 `/` 弹出命令菜单：

| 分类 | 命令 | 效果 |
|---|---|---|
| 标题 | `/标题1` `/标题2` `/标题3` | 对应级别标题 |
| 列表 | `/列表` `/有序` `/待办` | 各种列表 |
| 格式 | `/引用` `/代码` `/Mermaid` `/数学` `/脚注` `/分割线` | 块级格式 |
| 行内 | `/加粗` `/斜体` `/高亮` `/删除线` `/行内代码` `/链接` | 行内格式 |
| 插入 | `/图片` `/视频` `/表格` | 媒体和表格 |
| AI | `/AI` | AI 写作助手 |

---

## 搜索和替换

按 `Ctrl/Cmd + H` 打开搜索替换面板，在当前笔记中搜索和替换文字。

---

## 笔记锁定

点击工具栏 🔒 图标锁定笔记，防止误改。解锁后才能继续编辑。
