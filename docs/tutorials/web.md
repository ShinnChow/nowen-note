# Web 端使用指南

> 在浏览器中直接使用 nowen-note，无需安装任何客户端。

---

## 访问方式

在浏览器中输入你的 nowen-note 服务器地址即可访问：

- 自部署：`http://<服务器IP>:3001`
- 在线体验：[https://note.nowen.cn/](https://note.nowen.cn/)

---

## 推荐浏览器

| 浏览器 | 支持 |
|---|---|
| Chrome 90+ | ✅ 完整支持 |
| Edge 90+ | ✅ 完整支持 |
| Firefox 90+ | ✅ 完整支持 |
| Safari 15+ | ✅ 完整支持 |

---

## 功能对比

Web 端支持所有核心功能：

- ✅ 富文本和 Markdown 编辑器
- ✅ 文档树和笔记管理
- ✅ AI 功能（需配置）
- ✅ 思维导图
- ✅ 标签、收藏、搜索
- ✅ 说说和待办
- ✅ 文件管理
- ✅ 分享和协作
- ✅ 版本历史
- ✅ 暗色/亮色主题

---

## 快捷键

Web 端支持以下全局快捷键：

| 快捷键 | 效果 |
|---|---|
| `Ctrl/Cmd + K` | 命令面板 |
| `Ctrl/Cmd + N` | 新建笔记 |
| `Ctrl/Cmd + Shift + N` | 新建笔记本 |
| `Ctrl/Cmd + /` | 搜索 |
