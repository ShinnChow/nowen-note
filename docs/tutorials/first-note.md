# 创建你的第一篇笔记

> 从零开始，写出你的第一篇 nowen-note 笔记。

---

## 前提

确保已安装并登录 nowen-note。如果没有，请先参考 [5 分钟快速上手](./quick-start.md)。

---

## 第一步：创建笔记本

1. 看左侧边栏「笔记本」区域上方，点击 **＋** 按钮
2. 输入名称，例如「随想笔记」
3. 按回车

[截图：创建笔记本]

---

## 第二步：新建笔记

1. 点击刚创建的笔记本
2. 中间笔记列表出现「新建笔记」按钮，点击

[截图：新建笔记按钮]

---

## 第三步：写标题和正文

在标题栏输入标题，在正文区域输入内容。

试试这些操作：

- **加粗**：选中文字 → `Ctrl/Cmd + B`
- **标题**：新行输入 `## ` 后跟文字
- **列表**：新行输入 `- ` 后跟内容
- **斜杠命令**：新行输入 `/` 弹出命令菜单

[截图：编辑中的笔记]

---

## 第四步：自动保存

nowen-note 自动保存内容。底部状态栏显示：「已保存」/「保存中...」/「保存失败」。

---

## 第五步：加标签

编辑器顶部标签栏 → 输入标签名 → 按回车创建。

---

## 第六步：收藏

点击工具栏 ⭐ 按钮收藏。之后在左侧「收藏」入口快速找到。

---

## 常用操作速查

| 操作 | 方法 |
|---|---|
| 创建笔记本 | 侧边栏 ＋ 按钮 |
| 新建笔记 | 选笔记本 → 新建笔记按钮 |
| 加粗 | `Ctrl/Cmd + B` |
| 斜体 | `Ctrl/Cmd + I` |
| 插入链接 | `Ctrl/Cmd + K` |
| 插入图片 | 拖拽图片到编辑器 |
| 斜杠命令 | 输入 `/` |
| 搜索 | 侧边栏搜索框 |
| 收藏 | 工具栏 ⭐ 按钮 |
