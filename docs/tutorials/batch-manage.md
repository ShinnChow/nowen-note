# 笔记排序和批量管理

> 批量移动、删除、AI 标签，高效管理大量笔记。

---

## 排序方式

笔记列表支持多种排序：

| 排序方式 | 说明 |
|---|---|
| 手动排序 | 拖拽笔记调整顺序（默认） |
| 按更新时间 | 最近修改的排在前面 |
| 按创建时间 | 最新创建的排在前面 |
| 按标题 | 按标题字母/拼音排序 |

点击笔记列表顶部的排序按钮切换。排序偏好会自动保存。

---

## 置顶笔记

重要笔记可以置顶，始终显示在列表最上方：

1. 右键笔记 →「置顶」
2. 或在编辑器工具栏点击 📌 按钮

取消置顶：右键笔记 →「取消置顶」

---

## 批量选择

### 多选笔记

- **Ctrl/Cmd + 点击**：逐个选择/取消
- **Shift + 点击**：选择范围（从上次点击到本次点击之间的所有笔记）

选中后，笔记列表顶部出现批量操作栏。

---

## 批量操作

### 批量移动

1. 选中多篇笔记
2. 右键 →「移动到…」
3. 选择目标笔记本

### 批量删除

1. 选中多篇笔记
2. 右键 →「移入回收站」

### 批量 AI 标签

1. 选中多篇笔记
2. 右键 →「AI 标签」
3. AI 为每篇笔记生成标签

### 批量 AI 归类

1. 选中多篇笔记
2. 右键 →「AI 归类」
3. AI 建议移动到哪个笔记本

> 💡 单次最多处理 30 篇笔记。

---

## 导入笔记

nowen-note 支持从多种来源导入笔记：

| 来源 | 说明 |
|---|---|
| Word 文档 | 右键笔记本 →「导入 Word 文档」 |
| 公众号文章 | 右键笔记本 →「导入公众号文章」 |
| URL 导入 | 导入网页内容 |
| 有道云笔记 | 设置 → 数据管理 → 导入 |
| 小米笔记 | 设置 → 数据管理 → 导入 |
| iCloud 备忘录 | 设置 → 数据管理 → 导入 |
| OPPO 便签 | 设置 → 数据管理 → 导入 |

---

## 导出笔记

### 单篇导出

右键笔记 → 选择导出格式：

| 格式 | 说明 |
|---|---|
| Markdown | 导出为 .md 文件 |
| PDF | 导出为 PDF 文件 |
| 图片 | 导出为 PNG 图片 |
| Word | 导出为 .docx 文件 |

### 整个笔记本导出

右键笔记本 →「导出为 Markdown」，导出该笔记本下所有笔记。

---

## 常见问题

### Q：批量操作很慢？

AI 批量操作需要逐篇分析内容，多篇笔记会花一些时间。

### Q：导入的笔记格式不对？

不同来源的笔记格式可能有差异。导入后建议检查一下格式。
